MSXDOS でソフトを選択,起動するランチャ（v0.10）【暫定公開】

RUN.COM


<概要>
MSXDOS でソフトを選択,起動するランチャ
キーボード, JOYPAD（port1）のいずれかで操作可能
ファイル数は 40ファイルまで （超過分は一覧表示しない）

<対象>
 *.BAT：DOS の BAT ファイル、そのまま実行する
 *.COM：DOS の COM ファイル、そのまま実行する
 *.BAS：BASIC ファイル、「BASIC name.BAS」として実行する

<環境>
MSX-DOS1が起動すること（DOS2、漢字モードは想定外）
キーボード, JOYPAD（port1）のいずれかが操作できること

<使い方>
・RUN.COM を起動する
・カーソルでファイルを選択し実行、もしくは終了してDOSへ戻る
・キー操作 ［］はキーボード、{} は JOYPAD1
　・十字キー　　：カーソル移動　上下は +-1、左右は +-4 ずつ移動する
　・[ESC],{B} 　：終了
　・[SPACE],{A} ：選択したファイルを実行


<お断り>
・実験的にプログラムを組んでいるため、動作保証は出来かねます。
・所持していないため、色々なテープソフトので動作検証ができていません。
・起動しない、途中で止まるなどが起こる可能性があります。
・予告なくプログラムの修正や削除、公開中止をすることがあります。
